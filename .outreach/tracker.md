# Submission tracker

Log every submission so we don't double-submit and can correlate
inbound traffic / backlinks against outreach effort.

## Format

`<date>` — `<channel>` — `<status>` — `<URL of submission, if public>` — `<notes>`

Status values: `submitted`, `accepted`, `merged`, `rejected`, `pending review`, `live`.
